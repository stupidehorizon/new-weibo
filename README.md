# new-weibo
## 简介
node+express+ejs+mongodb 打造的一个实时分享平台，可实现注册，登录，发布图文，点赞，评论功能。
## 工具
- 前端：ejs模板，less。
- 后端：node,express,mongodb数据库。
- 自动化工具 : gulp。
- 服务器：阿里云，nginx，linux。

## 功能
1. 注册，登录。
2. 微博发布。
3. 图文发布。
4. 头像修改。
5. 昵称修改。
